Паттерн проектирования «Одиночка» (Singleton) — один из порождающих пат-
тернов проектирования и один из простейших паттернов в языке Java. Согласно
этому паттерну класс использует для всех вызовов один и тот же объект, то есть
создание экземпляров класса ограничивается одним объектом с предоставлением
глобальной точки доступа к данному классу. Это значит следующее: класс отвеча-
ет за создание объекта, а также гарантирует, что для всех клиентских обращений
к этому объекту будет создан лишь один объект. Данный класс не допускает не-
посредственного создания своих объектов. Получить его экземпляр можно только
с помощью предоставляемого им статического метода.
